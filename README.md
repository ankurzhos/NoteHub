NoteHub 📝

A React + TypeScript notes management application with full CRUD functionality and server-side pagination.

✨ Features

🆕 Create, delete, and search notes
📄 Server-side pagination for efficient data handling
⏱ Debounced search input for better performance
🔒 Token-based API authentication
✅ Form validation with Formik + Yup
🖼 Reusable Modal component via React Portal
💻 Fully typed API integration with TypeScript


🛠 Tech Stack

⚛️ React
📝 TypeScript
🔄 TanStack Query (React Query)
🌐 Axios
🖊 Formik
✅ Yup
📑 React Paginate
⏳ use-debounce
🎨 CSS Modules

🏗 Architecture

📂 /types — Type definitions
📡 /services — API layer
🧩 /components — UI components
⚖️ Clear separation of concerns between UI and data layer
📊 Server state management using React Query
